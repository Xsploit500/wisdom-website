# Wisdom Ariagbofo — Personal Website

Personal website for Wisdom Ariagbofo, a CS + Finance graduate from the University of Arkansas at Pine Bluff. Built as a single self-contained HTML file with no dependencies or build tools required.

## Stack

- HTML, CSS, JavaScript
- Google Fonts (Syne, DM Mono, Lora)
- Hosted on Vercel

## Features

- Multi-page navigation (Home, About, Work, Projects, Contact)
- Light / dark mode toggle with localStorage persistence
- Animated hero photo with slow zoom and entrance transitions
- About page photo slide-in animation
- Fully responsive layout across desktop, tablet, and mobile
- All images embedded as base64 — no external assets needed

## Deployment

Deployed via [Vercel](https://vercel.com). Any push to the `main` branch triggers an automatic redeployment.

## Local Preview

No build step needed. Just open `wisdom_final.html` in any browser.
